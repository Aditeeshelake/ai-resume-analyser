# ai-resume-analyser

This project analyzes resumes using NLP and compares them with job descriptions.

## Features
- Resume PDF upload
- Skill extraction
- Job description matching
- Match score percentage
- Suggestions for improvement

## Tech Stack
- Python
- Streamlit
- NLP (NLTK)
- Scikit-learn

## How to Run
pip install -r requirements.txt
streamlit run app.py
